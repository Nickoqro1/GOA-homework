HTML

HTML არის სტანდარტული მარკირების ენა ვებ გვერდების შესაქმნელად. HTML აღწერს ვებ გვერდის სტრუქტურას. HTML შედგება ელემენტების სერიისგან.

CSS

CSS არის „კასკადური სტილის ფურცლების“ აბრევიატურა. CSS არის კომპიუტერული ენა ვებ გვერდების განლაგებისა და სტრუქტურირებისთვის.

javascript

JavaScript არის ტექსტზე დაფუძნებული პროგრამირების ენა, რომელიც გამოიყენება როგორც კლიენტის, ასევე სერვერის მხარეს, რომელიც საშუალებას გაძლევთ გახადოთ ვებ გვერდები ინტერაქტიული.
